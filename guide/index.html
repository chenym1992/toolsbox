Fe-Toolsbox #

versionlicense

前端业务常用工具库

目的：高效率完成前端业务代码

安装 🔧 #

使用 npm、yarn、pnpm 安装

shell
npm i fe-toolsbox -D
yarn add fe-toolsbox -D
pnpm add fe-toolsbox -D

快速上手🔑 #

js
import { randomNumber, randomUID } from "fe-toolsbox";
randomNumber();
// 11 [0-100]
randomNumber(0,1);
// 0 | 1
randomUID()
//'9f235a5f-27e8-4e72-a854-f5b1fd027427'

API 文档📦 #

array #

  • array 数组
    • sum 数组求和
    • average 数组求平均数
    • uniq 数组去重

dom #

  • class dom class
    • hasClass 是否存在指定class
    • addClass 添加class
    • toggleClass 切换class

functions #

  • byteToString 字节数组转字符串
  • stringToByte 字符串转字节数组
  • throttle 节流函数
  • debounce 防抖函数
  • EventEmitter EventEmitter
  • convertColors 转换颜色格式
    • hexToRgb 颜色 hex 转 Rgb
    • hexToRgba 颜色 hex 转 Rgba
    • rgbToHex 颜色 rgb 转 hex
    • rgbaToHex 颜色 rgba 转 hex
  • random 随机
    • randomNumber 生成随机数,默认0-100
    • randomUID 随机生成一个UID
  • network 转换颜色格式
    • isHttps 判断https
    • isHttp 判断http
    • isOnline 判断网络状态

regexp #

  • color 颜色格式判断
    • isColor 判断是否是颜色表达式
    • isHexColor 判断 hex 颜色表达式
    • isRgbColor 判断 RGB 颜色表达式
    • isRgbaColor 判断 RGBA 颜色表达式
  • string 字符串格式判断
    • isIphoneNumber 宽松判断国内手机号
    • isTelNumber 判断座机号码（带分机号码）
    • isIdCard 判断身份证号
    • isValidIdCard 判断身份证是否有效
    • isValidIdCard 判断身份证是否有效
    • isEmail 判断邮箱格式
    • isUrl 判断Url格式
    • isEnglish 判断英文格式
    • isChinese 判断中文格式
    • isNumberPlate 判断车牌号格式
    • isBase64 判断base64格式
    • isBankCardNumber 判断银行卡号(包含对公对私)格式
    • isStrongPwd 强密码格式校验
    • checkPwdStrength 检测密码强度

time #

  • makeDurationPretty 美化时长显示
  • dateFormat 时间格式化

url #

  • getUrlParams 获取 Url 完整参数或指定字段参数
  • stringifyQueryString 简单对象序列化

utils #

  • classof 获取任意对象基本数据类型
  • isNumber 判断 number 类型
  • isString 判断 string 类型
  • isBoolean 判断 bool 类型
  • isUndefined 判断 undefined 类型
  • isNull 判断 null 类型
  • isSymbol 判断 symbol 类型
  • isBigInt 判断 bigint 类型
  • isObject 判断 object 类型
  • isFunction 判断 function 类型
  • isArray 判断 array 类型
  • isDate 判断 date 类型
  • isRegExp 判断 regexp 类型
  • isNaN 判断 NaN
  • isNil 判断 null||undefined
